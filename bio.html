Desenvolvimento Robô

Para o desenvolvimento do robô utilizamos básicamente uma placa Arduino Uno R3, que é uma placa de hardware livre Open Source

Ultilizamos algumas tecnologias voltadas especificamente para definir e programar o comportamento do robô sendo elas ARDUINO IDE, e uzamos o TinkerCad para criar os protótipo.

Fluxo de Trabalho

O maior desafio foi não ter tido base para confeccionar esse projeto, praticamente foi feito baseado em muita pesquisa

Do Your Self

Nosso fluxo de trabalho foi praticamente baseado em pequenas reuniões feitas pelo discord, onde definiamos melhorias e alinhavamos os afazeres
